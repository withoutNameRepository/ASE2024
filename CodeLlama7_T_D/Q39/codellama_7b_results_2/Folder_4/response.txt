 Here is a possible implementation of the `return_n_greatest_chars` function:

```python
def return_n_greatest_chars(string, n):
    # Create a list of characters from the input string
    char_list = list(string)
    # Sort the list in descending order based on ASCII values
    sorted_list = sorted(char_list, key=ord, reverse=True)
    # Return the first `n` elements of the sorted list
    return sorted_list[:n]
```
This function takes two arguments: a string of distinct characters and an integer `n`. It creates a list of characters from the input string, sorts the list in descending order based on ASCII values, and returns the first `n` elements of the sorted list.
