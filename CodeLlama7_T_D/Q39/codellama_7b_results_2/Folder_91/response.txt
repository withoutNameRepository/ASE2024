 ```
def return_n_greatest_chars(string, n=6):
    if len(string) < n:
        raise ValueError("Length of string is less than n")
    char_list = []
    for char in string:
        char_list.append((ord(char), char))
    char_list = sorted(char_list, reverse=True)[:n]
    return [char for ord_, char in char_list]
```
This code creates a function called `return_n_greatest_chars` that takes one argument, a string of distinct characters. The function sorts the characters in the string by their ASCII values and returns a list of exactly 6 characters with the greatest ASCII values. The sorting is done in descending order and the resulting list is returned.
Note: If the length of the input string is less than 6, this code will raise a `ValueError`.
