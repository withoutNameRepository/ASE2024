 ```
def return_binary_or_hexa(tuple):
a = tuple[0]
b = tuple[3]
sum = 0
for i in range(a + 1, b):
if i not in tuple:
sum += i
if sum % 2 == 0:
return hex(sum)[2:]
else:
return bin(sum)[3:]

```
The above code will return the binary representation of the sum if it is an odd number and the hexadecimal representation of the sum otherwise.
