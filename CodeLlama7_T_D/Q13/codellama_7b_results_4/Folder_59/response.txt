 Here's a possible implementation of the `find_second_largest_num` function in Python:
```
def find_second_largest_num(my_list):
    if len(my_list) < 2:
        return None

    sorted_list = sorted(my_list)

    for i in range(len(sorted_list)):
        if i > 0 and sorted_list[i] != sorted_list[i-1]:
            return sorted_list[i-1]

    return None
```
This function takes a list of distinct numbers as input, sorts the list in ascending order, and then iterates over the sorted list to find the second largest element. If no such element is found, the function returns `None`.
