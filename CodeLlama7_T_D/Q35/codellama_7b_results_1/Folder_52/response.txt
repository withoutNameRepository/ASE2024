 Here is a possible implementation of the `remove_repeat_chars` function:
```python
def remove_repeat_chars(s):
    # Create a new string with the same characters as s, but without any repeated characters
    new_string = ""
    for i in range(len(s)):
        if s[i] not in new_string:
            new_string += s[i]
    return new_string
```
This function first creates a new string `new_string` with the same characters as the original string `s`, but without any repeated characters. It then iterates over each character in `s`, checking if it is already present in `new_string`. If not, it adds the character to `new_string`. Finally, the function returns `new_string`.
