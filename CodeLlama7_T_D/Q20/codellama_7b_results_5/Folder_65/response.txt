 Here's a possible implementation of the `find_n_th_smallest_num` function:
```python
def find_n_th_smallest_num(my_list, n):
    # Sort the list in ascending order
    my_list.sort()
    # Return the n-th smallest number
    return my_list[n - 1]
```

This implementation first sorts the input `my_list` in ascending order using the `sort()` method. Then, it returns the `n`-th smallest number from the sorted list by indexing into it with the formula `my_list[n - 1]`.
Note that this function assumes that the input `my_list` is a valid list of distinct numbers, and that the index `n` is a valid integer between 1 and the length of the list (inclusive). If these assumptions are not met, the function may raise an exception or produce incorrect results.
