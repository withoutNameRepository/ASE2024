 Here is a possible implementation of the `find_n_th_smallest_num` function:
```python
def find_n_th_smallest_num(numbers, n):
    # Sort the list in ascending order
    numbers.sort()

    # Return the n-th smallest number
    return numbers[n - 1]
```
This code takes two arguments: a list of distinct numbers and an integer `n` that represents the position of the smallest number to be returned. It first sorts the input list in ascending order, then returns the element at position `n - 1`.
