 ```
def if_contains_anagrams(strings):
    anagram_pairs = 0
    for i in range(len(strings) - 1):
        for j in range(i + 1, len(strings)):
            if sorted(strings[i].lower()) == sorted(strings[j].lower()):
                anagram_pairs += 1
                if anagram_pairs >= 73:
                    return True

    return False
```
Note that this function assumes that the input is a list of strings. If the input is a single string, it must be split into a list of individual words before being passed to this function.
