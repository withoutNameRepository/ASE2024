 ```
def if_contains_anagrams(string_list):
    # Initialize variables to keep track of anagram pairs
    anagram_pairs = 0
    for i in range(len(string_list) - 1):
        for j in range(i + 1, len(string_list))):
            # Check if the current strings are anagrams
            if sorted(string_list[i]) == sorted(string_list[j])):
                # Increment the number of anagram pairs found
                anagram_pairs += 1
    return anagram_pairs >= 153
```
This function takes a list of strings as its argument. It then iterates over each pair of adjacent strings in the input list and checks if they are anagrams. If they are, the function increments a counter variable to keep track of the number of anagram pairs found. Finally, the function returns True if the number of anagram pairs found is greater than or equal to 153; otherwise it returns False.
Note that this function only considers anagrams of lengths at least three, as specified in the problem statement.
