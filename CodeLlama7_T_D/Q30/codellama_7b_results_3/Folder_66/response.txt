 Here is the Python code for the function `insert_before_character` that takes a string and returns a modified string with character `'k'` inserted before any occurrences of character `'k'`:
```python
def insert_before_character(string):
    new_string = ""
    for i in range(len(string)):
        if string[i] == "k":
            new_string += "k" + string[i]
        else:
            new_string += string[i]
    return new_string
```
