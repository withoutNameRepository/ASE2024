def return_vowels(my_string):
    vowels = ['a', 'e', 'i', 'o', 'u']
    start_index = 273
    end_index = 275
    filtered_chars = []
    for char in my_string[start_index:end_index]:
        if char.isalpha() and char.lower() in vowels:
            if char.lower() > 'a' and char.lower() <= 'o':
                filtered_chars.append(char)
    return filtered_chars

If the character meets these criteria, it then checks if its lowercase form (`char.lower()`) is greater than the letter `'a'` and smaller than or equal to the letter `'o'`. If this condition is met, the function adds the character to the `filtered_chars` list.
