 ```
def return_vowels(my_string):
    vowels = []
    for i in range(31, 37):
        if my_string[i] > ';' and my_string[i] <= 't':
            vowels.append(my_string[i])
    return vowels
```
This code defines a function called `return_vowels` that takes one argument, a string. The function first initializes an empty list to store the vowel characters. Then it iterates over the characters in the specified range (31-37) of the input string. If the current character is a vowel (i.e., `my_string[i] > ';' and my_string[i] <= 't'`), it is added to the list of vowels. Finally, the function returns the list of vowels.
