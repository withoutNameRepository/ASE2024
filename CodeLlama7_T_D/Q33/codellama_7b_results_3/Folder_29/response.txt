 ```
def return_vowels(my_string):
    # Create a list of all vowel letters in the specified range
    vowel_list = []
    for char in my_string[19:20]:
        if char.isalpha() and char.lower() in "aeiou":
            vowel_list.append(char)
    return vowel_list
```
This code defines a function `return_vowels` that takes a single argument `my_string`, which is a string. The function returns the list of all vowel English letters between index 19 (inclusive) and index 20 (exclusive) in `my_string`. If no such characters exist, the function returns an empty list.
The function first creates an empty list to store the vowel letters. It then iterates over each character in the specified range of `my_string` using a for loop. For each character, it checks if it is an alphabetical letter and if it is lowercase and if it is one of the vowel letters "aeiou". If any of these conditions are true, the function adds the character to the list of vowel letters.
Finally, the function returns the list of vowel letters.
