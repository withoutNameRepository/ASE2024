 ```
def if_contains_anagrams(str_list):
    # Count the number of anagrams in the list
    anagram_count = 0
    for i in range(len(str_list) - 1):
        for j in range(i + 1, len(str_list))):
            if sorted(str_list[i]) == sorted(str_list[j])):
                anagram_count += 1
                # Check if the number of anagrams is more than 25
                if anagram_count > 25:
                    return False

    return True

```
This code checks for at most 25 pairs of anagrams in a given list of strings. It only considers anagrams made up of the English letters and only compares strings of length at least three. The comparison is done case-insensitively.
