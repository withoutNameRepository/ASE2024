def if_contains_anagrams(my_list):
    # Create a set of all anagrams in the list
    anagram_set = {i for i in my_list if ''.join(sorted(i)) == ''.join(sorted(j))) for j in my_list}

It then checks the number of pairs of anagrams in the set using the len() method, and returns true if there are at most 8 pairs of anagrams, and false otherwise.
