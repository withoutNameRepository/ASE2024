def if_contains_anagrams(my_list):
    # Count the number of anagrams in the list
    anagram_count = 0
    for i in range(len(my_list) - 1)):
        for j in range(i + 1, len(my_list))):
            if sorted(my_list[i]) == sorted(my_list[j]):
                anagram_count += 1
    # Check if the number of anagrams is less than or equal to 401
    return anagram_count <= 401

If there are at most 401 pairs of anagrams in the given list, the function returns `True`. Otherwise, it returns `False`. The function only considers the anagrams of the English letters, and it only considers the anagrams of the length of at least three. The comparison is done case-insensitively.
