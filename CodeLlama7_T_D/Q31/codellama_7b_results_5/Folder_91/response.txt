 ```
def if_perfect_num(my_list):
    # Check if the list has at least 7 elements
    if len(my_list) < 7:
        return False

    # Get the integer at index 6
    num = my_list[6]

    # Check if the integer is a perfect number
    if num == sum(range(1, num))) {
        return True
    } else {
        return False
    }
```
This code defines a function called `if_perfect_num` that takes a list of positive integers as input. The function first checks if the length of the list is at least 7, and if not, returns `False`. Otherwise, it gets the integer at index 6 in the list, and uses this value to check if it is a perfect number or not.
