 Here is a possible implementation of the `if_perfect_num` function:
```python
def if_perfect_num(my_list):
    # Check that the list has at least 8 elements
    if len(my_list) < 8:
        return False

    # Get the number at index 8 of the list
    num = my_list[8]

    # Check if the number is a perfect number
    if num == sum(range(1, num))) {
        return True
    } else {
        return False
    }
```
