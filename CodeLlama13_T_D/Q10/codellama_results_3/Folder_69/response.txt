```python
def all_odd_ints_exclusive(my_list):
    return [x for x in my_list if x % 2 != 0 and 68 < x < 99]
```
This function uses a list comprehension to iterate over the elements of `my_list` and check whether each element is an odd integer between indices 68 and 99, both exclusive. If no such integers exist, it returns an empty list.
