```python
def all_odd_ints_exclusive(numbers):
    return [number for number in numbers if number % 2 != 0 and number < max(numbers)]
```
This function uses a list comprehension to iterate over the input `numbers` list, and returns a new list of all odd integers from index 0 to index 1, both exclusive. If no odd integers exist in the specified range, the function returns an empty list.
