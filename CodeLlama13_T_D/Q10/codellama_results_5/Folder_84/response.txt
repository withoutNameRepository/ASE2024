```
def all_odd_ints_exclusive(my_list):
    return [x for x in my_list if 50 < x < 92 and x % 2 == 1]
```
This function uses a list comprehension to iterate over the input list `my_list`, checking each element `x` to see if it is an odd integer between indices 50 and 92, both exclusive. If no such integers exist in the specified range, an empty list will be returned.
