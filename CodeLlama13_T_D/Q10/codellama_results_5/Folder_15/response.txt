```python
def all_odd_ints_exclusive(my_list):
    return [x for x in my_list if x % 2 != 0 and x <= 3]
```
This function uses a list comprehension to iterate over the elements of `my_list`, and returns only the odd integers from index 0 to index 3, both exclusive. The condition `x % 2 != 0` checks if the element is odd, and \begin{code}
x <= 3
\end{code} checks that it is within the specified range.
