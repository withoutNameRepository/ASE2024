```
def all_odd_ints_exclusive(my_list):
    return [x for x in my_list if x % 2 == 1 and 19 < x < 94]
```
This function takes a list of integers as input, and returns a new list containing only the odd integers from index 19 to index 94, both exclusive. If no odd integers exist in the specified range, it returns an empty list.
