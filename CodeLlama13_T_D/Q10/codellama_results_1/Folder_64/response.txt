```
def all_odd_ints_exclusive(my_list):
    return [x for x in my_list if x % 2 != 0 and 0 <= x < 5]
```
This function uses list comprehension to iterate over the input `my_list` and returns a new list with all odd integers from index 0 to index 4, both exclusive. If no odd integers exist in the specified range, an empty list is returned.
