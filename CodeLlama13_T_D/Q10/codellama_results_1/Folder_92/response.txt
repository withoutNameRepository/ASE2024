```
def all_odd_ints_exclusive(my_list):
    return [x for x in my_list if x % 2 == 1 and x <= 1]
```
This function uses a list comprehension to iterate over the elements of `my_list` and returns only those that are odd numbers less than or equal to 1. The use of the `and` operator ensures that the function will return an empty list if no odd integers exist in the specified range.
