```python
def all_odd_ints_exclusive(my_list):
    return [x for x in my_list if x % 2 != 0 and 389 < x < 748]
```
This code defines a function called `all_odd_ints_exclusive` that takes a list of integers as input. The function returns a new list containing all the odd integers from index 389 to index 748, both exclusive. If no odd integers exist in the specified range, an empty list is returned.

The code uses a list comprehension to create a new list that contains only the odd integers in the input list. The `x % 2 != 0` condition checks if `x` is odd, and the `389 < x < 748` condition checks if `x` is within the specified range.
