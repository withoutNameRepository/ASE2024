```
def all_odd_ints_exclusive(my_list):
    return [x for x in my_list[57:84] if x % 2 == 1]
```
This function uses list comprehension to iterate over the specified range of indices (from index 57 to index 84, both exclusive) and returns a new list containing all the odd integers from that range. If no odd integers exist in the specified range, an empty list is returned.
