```python
def all_odd_ints_exclusive(lst):
    return [x for x in lst if 30 < x < 200 and x % 2 == 1]
```
This function uses a list comprehension to iterate over the input list, `lst`, and returns a new list containing all odd integers from index 30 to index 200, both exclusive. If no odd integers exist in the specified range, an empty list is returned.
