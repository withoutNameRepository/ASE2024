```
def sum_ints_div_by_either_nums(numbers):
    result = 0
    for i in range(80, 201):
        if numbers[i] % 10 == 0 or numbers[i] % 1000 == 0:
            result += numbers[i]
    return result
```
This function takes a list of integers as input and returns the sum of all integers divisible by either 10 or 1000 from index 80 to index 200, both inclusive. If no such integers exist in the specified range, the function returns 0.
