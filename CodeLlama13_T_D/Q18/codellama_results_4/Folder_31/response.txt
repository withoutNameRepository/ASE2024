```
def sum_ints_div_by_either_nums(my_list):
    return sum(x for i, x in enumerate(my_list) if (i >= 56 and i <= 88 and (x % -59 == 0 or x % -79 == 0)))
```
This function uses a list comprehension to iterate over the elements of `my_list` from index 56 to 88, checking whether each element is divisible by either -59 or -79. The result is then summed using the built-in `sum()` function. If no such integers exist in the specified range, the function returns 0.
