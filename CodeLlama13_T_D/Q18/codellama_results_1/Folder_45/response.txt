```
def sum_ints_div_by_either_nums(arr):
    result = 0
    for i in range(30, 201):
        if arr[i] % -115 == 0 or arr[i] % 115 == 0:
            result += arr[i]
    return result
```
This function takes a list of integers as input and returns the sum of all integers divisible by either -115 or 115 from index 30 to index 200, both inclusive. If no such integers exist in the specified range, it will return 0.
