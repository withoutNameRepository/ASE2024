```python
def sum_ints_div_by_either_nums(my_list):
    result = 0
    for i in range(7, len(my_list)):
        if my_list[i] % 5 == 0 or my_list[i] % 7 == 0:
            result += my_list[i]
    return result
```
This function iterates over the list from index 7 to the end, and for each element it checks if it is divisible by either 5 or 7. If it is, it adds the element to the `result` variable. Finally, it returns the `result`. If no such integers exist in the specified range, the function returns 0.
