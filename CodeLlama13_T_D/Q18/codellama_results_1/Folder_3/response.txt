```python
def sum_ints_div_by_either_nums(my_list):
    result = 0
    for i in range(38, 52+1):
        if my_list[i] % -41 == 0 or my_list[i] % -47 == 0:
            result += my_list[i]
    return result
```
This function takes a list of integers as input and returns the sum of all integers divisible by either -41 or -47 from index 38 to index 52, both inclusive. If no such integers exist in the specified range, the function will return 0.
