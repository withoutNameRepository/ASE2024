```
def sum_ints_div_by_either_nums(nums):
    sum = 0
    for i in range(10, 29):
        if nums[i] % -61 == 0 or nums[i] % -64 == 0:
            sum += nums[i]
    return sum
```
This code creates a function `sum_ints_div_by_either_nums` that takes one argument, a list of integers `nums`, and returns the sum of all integers in the list that are divisible by either -61 or -64 from index 10 to index 28, both inclusive. If no such integers exist in the specified range, the function returns 0.
