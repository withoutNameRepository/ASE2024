```
def sum_ints_div_by_either_nums(nums):
    result = 0
    for i in range(25, 71):
        if nums[i] % 74 == 0 or nums[i] % 15 == 0:
            result += nums[i]
    return result
```
This function takes a list of integers as input and returns the sum of all integers divisible by either 74 or 15 from index 25 to index 70, both inclusive. If no such integers exist in the specified range, it returns 0.
