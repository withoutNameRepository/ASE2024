```
def sum_ints_div_by_either_nums(my_list):
    result = 0
    for i in range(81, 87):
        if my_list[i] % -33 == 0 or my_list[i] % -62 == 0:
            result += my_list[i]
    return result
```

It then iterates through the indices from 81 to 86 (both inclusive) and checks if each element is divisible by either -33 or -62.
