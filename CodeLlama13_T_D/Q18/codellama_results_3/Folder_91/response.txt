```python
def sum_ints_div_by_either_nums(my_list):
    result = 0
    for i in range(5):
        if my_list[i] % 8 == 0 or my_list[i] % -3 == 0:
            result += my_list[i]
    return result
```
This function takes a list of integers as an argument and returns the sum of all integers divisible by either 8 or -3 from index 0 to index 4, both inclusive. If no such integers exist in the specified range, the function returns 0.

Example:
```python
my_list = [1, 2, 3, 4, 5, 6]
result = sum_ints_div_by_either_nums(my_list)
print(result) # Output: 9
```
