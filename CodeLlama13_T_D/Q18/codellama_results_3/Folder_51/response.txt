```
def sum_ints_div_by_either_nums(lst):
    return sum([i for i in lst[6:10] if i % -1 == 0 or i % -10 == 0])
```
This function takes a list of integers as input, and returns the sum of all integers divisible by either -1 or -10 from index 6 to index 9 (both inclusive). If no such integers exist in the specified range, the function returns 0.
