```python
def sum_ints_div_by_either_nums(nums):
    result = 0
    for i in range(2, 8):
        if nums[i] % -1 == 0 or nums[i] % -7 == 0:
            result += nums[i]
    return result
```
Note that this function uses the `range` function to iterate over the specified range of indices, and the `if` statement to check if each integer is divisible by either -1 or -7. If either condition is true, the current integer is added to the running total.
