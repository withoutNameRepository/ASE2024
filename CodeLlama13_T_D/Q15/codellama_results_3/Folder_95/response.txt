```python
def sum_odd_ints_inclusive(my_list):
    return sum([x for i, x in enumerate(my_list) if (i >= 82 and i <= 86) and x % 2 != 0])
```
This function first uses `enumerate` to iterate over the list of integers with their indices. It then uses a list comprehension to select only the odd integers from index 82 to index 86, both inclusive. Finally, it returns the sum of these selected integers using the `sum` function. If no odd integers exist in the specified range, the function will return 0.
