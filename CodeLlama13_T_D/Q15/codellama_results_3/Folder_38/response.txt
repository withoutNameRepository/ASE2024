```
def sum_odd_ints_inclusive(my_list):
    odd_sum = 0
    for i in range(22, 64):
        if my_list[i] % 2 != 0:
            odd_sum += my_list[i]
    return odd_sum
```
This function iterates through the list from index 22 to index 63 and adds the value of each odd integer to `odd_sum`. If no odd integers exist in the specified range, `odd_sum` will be returned as 0.
