```python
def sum_odd_ints_inclusive(my_list):
    return sum([x for i, x in enumerate(my_list) if i in range(28, 33) and x % 2 != 0])
```
This function uses a list comprehension to create a new list of odd integers from the specified range. The `sum` function is then used to calculate the total of these odd integers. If no odd integers exist in the specified range, the function returns 0.
