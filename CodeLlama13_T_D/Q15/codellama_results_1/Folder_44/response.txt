```
def sum_odd_ints_inclusive(my_list):
    odd_nums = []
    for i in range(13, 69):
        if my_list[i] % 2 == 1:
            odd_nums.append(my_list[i])
    return sum(odd_nums)
```
This function takes a list of integers as an argument and returns the sum of all odd integers from index 13 to index 68, both inclusive. If no odd integers exist in the specified range, the function will return 0.
