```
def sum_odd_ints_inclusive(my_list):
    odd_sum = 0
    for i in range(6, 9):
        if my_list[i] % 2 == 1:
            odd_sum += my_list[i]
    return odd_sum
```
This function takes a list of integers as input and returns the sum of all odd integers from index 6 to index 8, both inclusive. If no odd integers exist in the specified range, the function returns 0.

For example, if `my_list = [1, 2, 3, 4, 5, 6, 7, 8, 9]`, then `sum_odd_ints_inclusive(my_list)` would return `1 + 3 + 5 + 7 = 16`.

Note that this function assumes that the input list contains at least 9 elements, and that all elements in the specified range are integers. If either of these conditions is not met, the function will raise an error or return incorrect results.
