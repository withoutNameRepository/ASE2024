```python
def sum_odd_ints_inclusive(my_list):
    return sum(x for i, x in enumerate(my_list) if i >= 91 and i <= 99 and x % 2 != 0) or 0
```
This function uses a list comprehension to extract all odd integers from the specified range. The `enumerate` function is used to get the index of each element in the list, which allows us to check if it falls within the desired range. If no odd integers are found, the function returns 0.
