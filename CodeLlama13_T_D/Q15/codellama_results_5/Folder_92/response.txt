```python
def sum_odd_ints_inclusive(my_list):
    return sum([i for i in my_list[1:2] if i % 2 == 1])
```
This function uses a list comprehension to iterate over the elements of `my_list` from index 1 (inclusive) to index 2 (exclusive), and returns the sum of all odd integers in that range. If no odd integers exist, it will return 0.
