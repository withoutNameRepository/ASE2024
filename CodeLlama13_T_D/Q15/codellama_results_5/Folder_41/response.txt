```
def sum_odd_ints_inclusive(lst):
    return sum([i for i in lst[6:7+1] if i % 2 == 1]) or 0
```
This function takes a list of integers as input and returns the sum of all odd integers from index 6 to index 6, both inclusive. If no odd integers exist in the specified range, it returns 0.

The function uses a list comprehension to select only the odd integers in the specified range. The `sum()` function is then used to calculate the total of these odd integers. Finally, if the sum is zero, the function returns 0, otherwise it returns the calculated sum.
