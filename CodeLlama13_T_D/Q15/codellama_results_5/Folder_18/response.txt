def sum_odd_ints_inclusive(my_list):
    result = 0
    for i in range(35, 49+1):
        if my_list[i] % 2 == 1:
            result += my_list[i]
    return result

The function `sum_odd_ints_inclusive` takes one argument, a list of integers, and returns the sum of all odd integers from index 35 to index 49, both inclusive. If no odd integers exist in the specified range, the function should return 0. The function uses a for loop to iterate through the indices from 35 to 49 and checks if each element is an odd integer using the modulo operator.
