```python
def sum_odd_ints_inclusive(my_list):
    result = 0
    for i in range(8):
        if my_list[i] % 2 == 1:
            result += my_list[i]
    return result
```
This function takes a list of integers as an argument and returns the sum of all odd integers from index 0 to index 7, both inclusive. If no odd integers exist in the specified range, the function returns 0.

Here is an example usage:
```python
my_list = [1, 2, 3, 4, 5, 6, 7]
print(sum_odd_ints_inclusive(my_list)) # prints 9 (1 + 3 + 5 + 7)

my_list = [1, 2, 4, 6, 8]
print(sum_odd_ints_inclusive(my_list)) # prints 0
```
