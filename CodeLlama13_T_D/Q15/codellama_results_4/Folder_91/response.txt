```
def sum_odd_ints_inclusive(my_list):
    total = 0
    for i in range(6):
        if my_list[i] % 2 == 1:
            total += my_list[i]
    return total
```
This function takes a list of integers as input and returns the sum of all odd integers from index 0 to index 5, both inclusive. If no odd integers exist in the specified range, it returns 0.
