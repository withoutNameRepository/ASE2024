```python
def sum_odd_ints_inclusive(my_list):
    total = 0
    for i in range(14, 65):
        if my_list[i] % 2 != 0:
            total += my_list[i]
    return total
```
This function uses a for loop to iterate over the specified range of indices (14-64) and checks whether each integer is odd using the modulo operator. If an odd integer is found, its value is added to the total sum. The function returns 0 if no odd integers exist in the specified range.
