```
def sum_odd_ints_inclusive(my_list):
    return sum([num for num in my_list[29:201] if num % 2 != 0])
```
This function uses a list comprehension to create a new list containing only the odd integers between indices 30 and 200, both inclusive. The `sum()` function is then used to calculate the total of these odd integers. If no odd integers exist in the specified range, the function returns 0.
