```python
def sum_odd_ints_inclusive(my_list):
    odd_sum = 0
    for i in range(8):
        if my_list[i] % 2 == 1:
            odd_sum += my_list[i]
    return odd_sum
```
This function takes a list of integers as input, and returns the sum of all odd integers from index 0 to index 7, both inclusive. If no odd integers exist in the specified range, the function returns 0.

Example usage:
```python
print(sum_odd_ints_inclusive([1, 2, 3, 4, 5, 6, 7])) # Output: 16 (1 + 3 + 5 + 7)
print(sum_odd_ints_inclusive([2, 4, 6, 8, 10])) # Output: 0 (since there are no odd integers in the list)
```
