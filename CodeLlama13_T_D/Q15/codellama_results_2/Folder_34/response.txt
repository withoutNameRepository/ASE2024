```
def sum_odd_ints_inclusive(lst):
    return sum([i for i in lst[59:201] if i % 2 == 1])
```
This function uses a list comprehension to select all the odd integers from index 60 to index 200, both inclusive. The `sum` function is then used to calculate the sum of these integers. If no odd integers exist in the specified range, the function returns 0.
