```
def sum_odd_ints_inclusive(my_list):
    return sum([num for i, num in enumerate(my_list) if i >= 30 and i <= 87 and num % 2 != 0])
```
This function uses a list comprehension to filter out the odd integers from the input list using the `enumerate()` function to get the index and value of each element. The `sum()` function is then used to calculate the sum of the filtered elements. If no odd integers exist in the specified range, 0 is returned.
