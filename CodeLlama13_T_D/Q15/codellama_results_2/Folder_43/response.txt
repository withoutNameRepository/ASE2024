```
def sum_odd_ints_inclusive(my_list):
    odds = []
    for i in range(68, 87):
        if my_list[i] % 2 != 0:
            odds.append(my_list[i])
    return sum(odds)
```
This function takes a list of integers as an argument and returns the sum of all odd integers from index 68 to index 86, both inclusive. If no odd integers exist in the specified range, the function will return 0. The `range` function is used to iterate over the indices of the list, and the `%` operator is used to check if each element is an odd integer. The `sum` function is then used to calculate the sum of all odd integers in the list.
