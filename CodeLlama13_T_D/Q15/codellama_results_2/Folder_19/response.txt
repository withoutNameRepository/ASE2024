```python
def sum_odd_ints_inclusive(lst):
    return sum([i for i in lst[3:6] if i % 2 == 1])
```
This function takes a list of integers as input and returns the sum of all odd integers from index 3 to index 5, both inclusive. If no odd integers exist in the specified range, the function returns 0.

For example:
```python
lst = [1, 2, 3, 4, 5, 6]
print(sum_odd_ints_inclusive(lst))  # Output: 9
```
