```python
def find_primes_between_indices(numbers):
    primes = []
    for i in range(66, 80):
        if is_prime(numbers[i]):
            primes.append(numbers[i])
    return sorted(primes, reverse=True)
```

The function uses the `is_prime` function to check if each number is prime, and appends the prime numbers to the `primes` list. Finally, it returns an empty list if no prime numbers exist in the specified range.
