```
def find_primes_between_indices(nums):
    primes = []
    for i in range(56, 58):
        if is_prime(nums[i]):
            primes.append(nums[i])
    return sorted(primes, reverse=True)
```
This code defines a function `find_primes_between_indices` that takes a list of integers as an argument and returns the sorted list of all prime numbers from index 56 to index 57, both inclusive. The returned list is in descending order. If no prime number exists in the specified range, the function returns an empty list.

The code uses a `for` loop to iterate over the indices 56 to 58 of the input list. For each index, it checks if the corresponding element is a prime number using the `is_prime` function, and appends it to the `primes` list if it is a prime number. Finally, it returns the sorted list of primes in descending order.

Note that this code uses the `sorted` function with the `reverse` parameter set to `True` to sort the list of primes in descending order.
