```python
def find_primes_between_indices(numbers):
    prime_nums = []
    for i in range(45, 81):
        if is_prime(numbers[i]):
            prime_nums.append(numbers[i])
    return sorted(prime_nums, reverse=True)
```
This function takes a list of integers as an argument and returns the sorted list of all prime numbers from index 45 to index 80, both inclusive, in descending order. If no prime number exists in the specified range, the function returns an empty list. The `is_prime()` function is not included in this solution.
