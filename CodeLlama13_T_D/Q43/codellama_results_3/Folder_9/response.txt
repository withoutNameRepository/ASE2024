```python
def find_primes_between_indices(numbers):
    primes = []
    for i in range(16, 72):
        if is_prime(numbers[i]):
            primes.append(numbers[i])
    return sorted(primes)
```
This code defines a function `find_primes_between_indices` that takes a list of integers as an argument and returns a sorted list of all prime numbers from index 16 to index 71, both inclusive. The function uses the `is_prime` function to check if each number in the input list is prime, and if it is, it adds it to the output list. Finally, the function sorts the output list and returns it. If no prime numbers exist in the specified range, the function returns an empty list.
