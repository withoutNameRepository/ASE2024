```
def find_primes_between_indices(numbers):
    primes = []
    for i in range(131, 815+1):
        if is_prime(numbers[i]):
            primes.append(numbers[i])
    return sorted(primes, reverse=True)
```
This function defines a helper function `is_prime` that checks whether a given number is prime or not. It then iterates over the list of numbers and appends each prime number to a new list called `primes`. Finally, it returns the sorted list of primes in descending order using the `sorted()` function with `reverse=True` parameter.
