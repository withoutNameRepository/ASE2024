```python
def find_primes_between_indices(numbers):
    primes = []
    for i in range(23, 60):
        if is_prime(numbers[i]):
            primes.append(numbers[i])
    return sorted(primes, reverse=True)

def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True
```

The returned list is in descending order. If no prime number exists in the specified range, the function returns an empty list.
