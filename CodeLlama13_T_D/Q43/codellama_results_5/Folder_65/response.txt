```
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True
```
This function uses the Sieve of Eratosthenes to check if a number is prime and then finds all the prime numbers in the specified range using a list comprehension.
