```python
def find_primes_between_indices(nums):
    primes = []
    for i in range(24, 83+1):
        if is_prime(nums[i]):
            primes.append(nums[i])
    return sorted(primes)
```
This function uses the `is_prime` function to check whether each number between indices 24 and 83 (inclusive) is prime, and if so, adds it to a list called `primes`. The `sorted` function is then used to sort the list of primes in ascending order. If no prime numbers exist in the specified range, an empty list is returned.
