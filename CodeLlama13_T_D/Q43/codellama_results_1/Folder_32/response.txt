```python
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True
```
This function takes a single integer `n` as an argument and returns `True` if `n` is prime, or `False` otherwise.
