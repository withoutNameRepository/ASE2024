```
def all_ints_div_by_both_two_nums(int_list):
    result = []
    for i in range(86, 90):
        if int_list[i] % -61 == 0 and int_list[i] % -79 == 0:
            result.append(int_list[i])
    return result
```
This function takes an integer list as input, iterates over the range of indices from 86 to 89 (both inclusive), checks if each element is divisible by both -61 and -79, and appends it to a result list if it meets the condition. Finally, it returns the result list or an empty list if no such integers exist in the specified range.
