```
def all_ints_div_by_both_two_nums(my_list):
    result = []
    for i in range(58, 81 + 1):
        if my_list[i] % 55 == 0 and my_list[i] % 10 == 0:
            result.append(my_list[i])
    return result
```
This function takes a list of integers as an argument, and returns a list of all integers in the list that are divisible by both 55 and 10 from index 58 to index 81, both inclusive. If no such integers exist in the specified range, the function returns an empty list.
