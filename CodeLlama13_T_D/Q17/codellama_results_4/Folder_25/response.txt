```python
def all_ints_div_by_both_two_nums(my_list):
    return [x for x in my_list[25:81] if (x % -20 == 0 and x % -26 == 0)]
```
Here is another solution using list comprehension with conditional expressions.
```python
def all_ints_div_by_both_two_nums(my_list):
    return [x for x in my_list[25:81] if (x % -20 == 0 and x % -26 == 0) or (x % -26 == 0 and x % -20 == 0)]
```
