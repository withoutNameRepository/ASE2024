```
def all_ints_div_by_both_two_nums(arr):
    return [x for x in arr[77:88] if x % 23 == 0 and x % 57 == 0]
```
This is a list comprehension that returns a new list containing all integers from the input list `arr` that are divisible by both 23 and 57. The range of indices to consider is specified as `[77:88]` (both inclusive). If no such integers exist, an empty list will be returned.
