```python
def insert_before_character(string, char):
    return "".join([char if c == "j" else c for c in string])
```
For example:
```python
print(insert_before_character("hello", "1")) # Output: "he1llo"
print(insert_before_character("hijkl", "1")) # Output: "he1ijkl"
print(insert_before_character("jjjj", "1")) # Output: "1jjjj"
```
