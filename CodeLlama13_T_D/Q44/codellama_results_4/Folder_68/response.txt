```
def composite_nums_between_indices(lst):
    return {x for x in lst[0:8+1] if len({2,3,5,7}[x]) != 4}
```
This function uses a set comprehension to find all numbers between indices 0 and 8 that are composite. It checks if the length of the intersection of the number with the set {2,3,5,7} is not equal to 4, which indicates that the number is composite. The result is returned as a set.
