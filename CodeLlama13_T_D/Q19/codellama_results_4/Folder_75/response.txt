```python
def all_ints_not_div_by_num(my_list):
    return [x for x in my_list[20:51] if x % -48 != 0]
```
This function uses a list comprehension to iterate over the elements of `my_list` from index 20 to index 51, both exclusive. For each element `x`, it checks if `x` is not divisible by `-48`. If it is not divisible, it adds `x` to the resulting list. Finally, it returns the resulting list.

Here is an example usage of the function:
```python
my_list = [1, 2, 3, 4, 5]
print(all_ints_not_div_by_num(my_list)) # Output: []
```
