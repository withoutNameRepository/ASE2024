```
def all_ints_not_div_by_num(my_list):
    result = []
    for i in range(2, len(my_list)):
        if my_list[i] % 2 != 0:
            result.append(my_list[i])
    return result
```
This function takes a list of integers as an argument and returns a list of all integers from index 2 to the end of the list that are not divisible by 2, both exclusive. If no such integers exist in the specified range, the function returns an empty list.
