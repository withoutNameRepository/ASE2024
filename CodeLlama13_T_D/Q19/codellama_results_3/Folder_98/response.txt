```
def all_ints_not_div_by_num(arr):
    res = []
    for i in range(6):
        if arr[i] % 13 != 0:
            res.append(arr[i])
    return res
```
This function takes a list of integers `arr` as input and returns a new list `res` that contains all the integers from index 0 to index 5 (both exclusive) in `arr` that are not divisible by 13. If no such integer exists, an empty list is returned.
