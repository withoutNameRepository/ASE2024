```
def all_ints_not_div_by_num(list):
    return [i for i in list[50:81] if i % 64 != 0]
```
This function takes a list of integers as an argument and returns the list of all integers not divisible by 64 from index 50 to index 81, both exclusive. If no such integers exist in the specified range, the function returns an empty list.
