```python
def all_ints_not_div_by_num(my_list):
    return [x for x in my_list[38:81] if x % 49 != 0]
```
This code defines a function `all_ints_not_div_by_num` that takes a list of integers as an argument. It then uses list comprehension to create a new list containing all the elements from index 38 to index 81, both exclusive, that are not divisible by 49. If no such elements exist in the specified range, it returns an empty list.
