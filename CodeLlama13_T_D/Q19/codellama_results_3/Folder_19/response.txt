```python
def all_ints_not_div_by_num(my_list):
    return [x for x in my_list[4:5] if x % -7 != 0]
```
This function takes a list of integers as an argument and returns a new list containing only the elements from index 4 to index 5 (exclusive) that are not divisible by -7. If no such elements exist in the specified range, it returns an empty list.
