```
def all_ints_not_div_by_num(my_list):
    return [x for x in my_list[1:6] if x % 2 != 0]
```
This function takes a list of integers as input and returns a new list containing only the elements from indices 1 to 6 (both exclusive) that are not divisible by 2. If no such elements exist, an empty list is returned.
