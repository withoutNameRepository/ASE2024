```
def all_ints_not_div_by_num(my_list):
    return [x for x in my_list[15:20] if x % 93 != 0]
```
This function takes a list of integers as input and returns a new list containing the elements from index 15 to index 20, both exclusive, that are not divisible by 93. If no such elements exist in the specified range, an empty list is returned.
