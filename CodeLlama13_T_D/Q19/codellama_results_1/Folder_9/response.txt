```
def all_ints_not_div_by_num(my_list):
    return [x for x in my_list[70:200] if x % -300 != 0]
```
This function takes a list of integers as input and returns the list of integers that are not divisible by -300 from index 70 to index 200, both exclusive. If no such integers exist in the specified range, it returns an empty list.
