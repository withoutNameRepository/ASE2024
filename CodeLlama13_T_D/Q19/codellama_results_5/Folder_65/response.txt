```
def all_ints_not_div_by_num(my_list):
    return [x for x in my_list[15:51] if x % -34 != 0]
```
This function takes a list of integers as input and returns a new list containing all the elements from index 15 to index 51, both exclusive, that are not divisible by -34. If no such integers exist in the specified range, an empty list is returned.
