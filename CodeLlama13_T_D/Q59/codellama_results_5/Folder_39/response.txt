```python
    for i in range(2, x+1):
```

```python
def is_prime(n):
    if n == 1 or n == 0:
        return False
    for i in range(2, int(n**0.5)+1):
        if n % i == 0:
            return False
    return True
```
Note: The `contains_zero` function can be defined as follows:
```python
def contains_zero(s):
    for c in s:
        if c == '0':
            return True
    return False
```
