```
def sum_of_divisors_in_range(n):
    divisors = []
    for i in range(13, 62+1):
        if n % i == 0:
            divisors.append(i)
    return sum(divisors) or 0
```
This function first defines an empty list `divisors` to store the positive divisors of `n`. Then it iterates over all integers from 13 to 62, inclusive, and checks if each integer is a divisor of `n` using the modulo operator. If so, it adds that integer to the `divisors` list. Finally, it returns the sum of the elements in the `divisors` list or 0 if the list is empty.
