```
def sum_of_divisors_in_range(n):
    if n <= 0: return 0
    divisors = []
    for i in range(1, 9):
        if n % i == 0 and i in range(1, 9):
            divisors.append(i)
    return sum(divisors) or 0
```
This function takes a positive integer `n` as input and returns the sum of all positive divisors of `n` that are between 1 and 8, both inclusive. If no such divisors exist in the specified range, it returns 0.
