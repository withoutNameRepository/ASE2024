```
def sum_of_divisors_in_range(n):
    divisors = []
    for i in range(13, 62+1):
        if n % i == 0:
            divisors.append(i)
    return sum(divisors) or 0
```
Note: The `or` operator is used to handle the case when there are no divisors in the specified range, and return 0 instead of an empty list.
