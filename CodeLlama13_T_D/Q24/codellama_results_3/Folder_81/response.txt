```python
def sum_of_divisors_in_range(n):
    divisors = []
    for i in range(3, 4+1):
        if n % i == 0:
            divisors.append(i)
    return sum(divisors) or 0
```
Note that the `sum` function returns a zero value if no items are provided to it. This means that if there are no divisors in the range, the function will return 0.
