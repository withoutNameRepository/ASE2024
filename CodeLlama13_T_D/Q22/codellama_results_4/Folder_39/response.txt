```python
def find_sum_of_multiples(n):
    return sum(range(1, 94) if i % n == 0 else 0 for i in range(1, 94))
```
This function uses a generator expression to iterate over the numbers from 1 to 93 and check if each number is a multiple of `n`. If it is, it yields that number, otherwise it yields 0. The `sum` function then sums up all the yielded values.
