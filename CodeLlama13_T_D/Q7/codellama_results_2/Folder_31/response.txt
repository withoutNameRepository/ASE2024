```
def all_even_ints_inclusive(my_list):
    return [x for x in my_list[75:89] if x % 2 == 0]
```
This is a list comprehension that iterates over the specified range of indices, checks whether each integer is even using the modulo operator `%`, and returns a new list containing only the even integers. If no even integers exist in the specified range, an empty list is returned.
