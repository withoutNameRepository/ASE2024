```python
import numpy as np
def submatrix_with_n_numbers(matrix):
    rows, cols = matrix.shape
    for i in range(rows-6):
        for j in range(cols-6):
            submat = matrix[i:i+7, j:j+7]
            if (submat == 77).all():
                return True
    return False
```
This function iterates over the rows and columns of the given matrix, and for each position, it checks whether a 7x7 submatrix contains all 77s. If such a submatrix is found, the function returns `True`. Otherwise, the function returns `False`.
