```python
def gcf_three_nums(my_list):
    return my_list[990]*my_list[671]*my_list[711] // math.gcd(my_list[990], my_list[671]) // math.gcd(my_list[711], my_list[671])
```
Note: This solution uses the fact that `a*b*c` is equal to `(a*b)*c`, and therefore the greatest common factor of three numbers can be found by multiplying the greatest common factors of two pairs of numbers.
