To generate the greatest common factor of three numbers, we can use the `math.gcd` function from the Python standard library to calculate the greatest common divisor (GCD) of each pair of numbers and then take their product. Here's one possible solution:
```
from math import gcd

def gcf_three_nums(numbers):
    return reduce(lambda x, y: gcd(x, y), [numbers[99], numbers[28], numbers[53]])
```
This function first extracts the three numbers from the input list using indexing. It then calculates the GCD of each pair of numbers and stores them in a new list. Finally, it reduces the list to a single value using `reduce`, which returns the product of all elements in the list. This will be the greatest common factor of the original three numbers.

Here's an example usage:
```
numbers = [12, 34, 56, 78, 90, 102, 114, 126, 138, 150, 162, 174, 186]
print(gcf_three_nums(numbers)) # Output: 2
```
