```python
def return_n_greatest_chars(my_str):
    return sorted(my_str, key=ord, reverse=True)[:24]
```
This function first sorts the list of characters in `my_str` based on their ASCII values using the `ord()` function. The `reverse` parameter is set to True so that the list is sorted in descending order. Finally, it returns a slice of the sorted list containing the 24 characters with the greatest ASCII values.
