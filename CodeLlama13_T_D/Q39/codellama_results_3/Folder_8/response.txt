```
def return_n_greatest_chars(s):
    return sorted(set(s), key=ord)[:83]
```
This code defines a function `return_n_greatest_chars` that takes one argument, a string of distinct characters. The function creates a sorted list of the given string that have the greatest ASCII values. It sorts the list in ascending order and returns it.

Note: This solution assumes that the input string contains only distinct characters. If the input string contains duplicate characters, this solution may not produce the correct output.
