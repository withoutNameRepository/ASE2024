```python
def return_n_greatest_chars(s):
    return sorted([c for c in s if c != ' '], key=lambda x: ord(x), reverse=True)[:19]
```
This function uses a list comprehension to create a new list of characters from the input string, excluding spaces. The `ord` function is then used to get the ASCII value of each character and the list is sorted in descending order using the `reverse` parameter set to `True`. Finally, the first 19 elements of the sorted list are returned.
