```python
def return_n_greatest_chars(string):
    char_list = sorted([char for char in string], key=ord)
    return char_list[-10:]
```
This function uses the `sorted` function to sort the list of characters based on their ASCII values. The `key` argument is used to specify that the sorting should be done using the `ord` function, which returns the ASCII value of a character. The resultant sorted list is then sliced to get the last 10 elements, which are the 10 characters with the greatest ASCII values in the input string.
