```python
def return_n_greatest_chars(s):
    return sorted(list(set(s)), key=lambda x: ord(x))[:72]
```
This function takes a string `s` as input and returns a list of exactly 72 characters that have the greatest ASCII values in ascending order. It first creates a set of distinct characters from the input string, then sorts the list using the `sorted()` built-in function with a key function that uses the `ord()` function to get the ASCII value of each character. Finally, it returns the first 72 characters of the sorted list.
