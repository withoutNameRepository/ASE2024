```
def return_n_greatest_chars(s):
    return sorted([c for c in s if c.isalpha()], key=ord, reverse=True)[:72]
```
This function takes a string `s` as input and returns a list of 72 characters from the input string that have the greatest ASCII values. The characters are extracted using the `sorted()` function with the `key` parameter set to `ord`, which sorts the characters in descending order based on their ASCII value, and the `reverse` parameter set to `True` to get the list in descending order.

Here's an example usage of the function:
```python
print(return_n_greatest_chars('hello world')) # Output: ['w', 'd', 'l', 'o', 'h', 'e']
```
