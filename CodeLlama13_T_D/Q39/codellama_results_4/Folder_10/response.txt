```
def return_n_greatest_chars(s):
    char_count = 88
    ascii_values = {char: ord(char) for char in set(s)}
    sorted_ascii_values = sorted(ascii_values.items(), key=lambda x: x[1], reverse=True)
    return [char for char, _ in sorted_ascii_values[:char_count]]
```
This function uses a dictionary comprehension to create a dictionary that maps each distinct character in the input string to its ASCII value. It then sorts the dictionary using the `sorted` function with a custom key that extracts the ASCII value of each character. Finally, it returns a list of the first 88 characters from the sorted list.
