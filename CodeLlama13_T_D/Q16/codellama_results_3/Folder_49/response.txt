```
def sum_even_ints_inclusive(my_list):
    result = 0
    for i in range(8, 81):
        if my_list[i] % 2 == 0:
            result += my_list[i]
    return result
```
This function uses a `for` loop to iterate over the list and checks whether each element is even using the modulo operator (`%`). If an element is even, it adds its value to the total sum. The function returns 0 if no even integers exist in the specified range.
