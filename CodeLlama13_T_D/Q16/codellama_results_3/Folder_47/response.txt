```
def sum_even_ints_inclusive(my_list):
    return sum(filter(lambda x: x % 2 == 0, my_list[37:77]))
```
This function uses the `sum` function to get the total of all even numbers in the list, and the `filter` function with a lambda function to filter out non-even integers. The function returns 0 if no even integers are found in the specified range.
