```
def sum_even_ints_inclusive(my_list):
    # Find all even integers in the specified range
    evens = [x for x in my_list[15:40] if x % 2 == 0]
    
    # If no even integers exist, return 0
    if not evens:
        return 0
    
    # Otherwise, sum all even integers and return the result
    return sum(evens)
```
Note that this function uses a list comprehension to find all even integers in the specified range. The `if` clause filters out any odd integers, leaving only the even ones in the resulting list `evens`. If `evens` is empty, it means no even integers exist in the specified range, and the function returns 0. Otherwise, the function sums up all even integers using the built-in `sum()` function and returns the result.
