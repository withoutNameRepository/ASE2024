```python
def sum_even_ints_inclusive(my_list):
    return sum(x for i, x in enumerate(my_list) if i >= 26 and i <= 52 and x % 2 == 0) or 0
```
This function uses a list comprehension to create a new list of even integers in the specified range. The `sum()` function is then called on this list, returning the sum of all elements. If no even integers exist in the specified range, the function returns 0.
