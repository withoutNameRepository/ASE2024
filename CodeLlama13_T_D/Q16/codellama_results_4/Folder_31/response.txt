```python
def sum_even_ints_inclusive(my_list):
    return sum([x for x in my_list[75:89] if x % 2 == 0])
```
Explanation:

* The function takes a list of integers as an argument, `my_list`.
* We use a list comprehension to iterate over the elements of `my_list` from index 75 to index 88 (both inclusive).
* For each element, we check if it is even using the modulo operator (`%`). If it is even, we add it to the sum.
* We return the sum of all even integers in the specified range. If no even integers exist in the specified range, we return 0.
