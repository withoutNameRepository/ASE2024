```
def sum_even_ints_inclusive(my_list):
    # Initialize an empty list to store the even integers
    even_ints = []
    
    # Iterate through the input list and check if each integer is even
    for i in range(26, 53):
        if my_list[i] % 2 == 0:
            even_ints.append(my_list[i])
            
    # Return the sum of all even integers
    return sum(even_ints)
```
This solution uses a list comprehension to find all the even integers in the specified range, and then calculates the sum using the built-in `sum()` function. If no even integers exist in the specified range, the function returns 0.
