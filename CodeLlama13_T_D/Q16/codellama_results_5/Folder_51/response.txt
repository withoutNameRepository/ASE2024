```python
def sum_even_ints_inclusive(my_list):
    total = 0
    for i in range(1,8+1):
        if my_list[i] % 2 == 0:
            total += my_list[i]
    return total
```
This function takes a list of integers as input and returns the sum of all even integers from index 1 to index 8 both inclusive. If no even integers exist in the specified range, it returns 0.
