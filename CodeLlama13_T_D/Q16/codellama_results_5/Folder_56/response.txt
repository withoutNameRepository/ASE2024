```
def sum_even_ints_inclusive(my_list):
    result = 0
    for i in range(len(my_list)):
        if my_list[i] % 2 == 0:
            result += my_list[i]
    return result
```
This function takes a list of integers as an argument and returns the sum of all even integers from index 0 to index 0, both inclusive. If no even integers exist in the specified range, the function returns 0.
