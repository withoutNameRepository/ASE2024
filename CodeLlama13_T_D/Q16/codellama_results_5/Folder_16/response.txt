```
def sum_even_ints_inclusive(numbers):
    total = 0
    for i in range(33, 37):
        if numbers[i] % 2 == 0:
            total += numbers[i]
    return total
```
This code defines a function called `sum_even_ints_inclusive` that takes one argument, a list of integers. The function returns the sum of all even integers from index 33 to index 36 (both inclusive). If no even integers exist in the specified range, the function returns 0.

The function uses a for loop to iterate over the specified range of indices (33 to 36) and checks if each number is even using the modulo operator (%). If a number is even, it adds it to the total sum. Finally, the function returns the total sum.
