```python
def sum_even_ints_inclusive(my_list):
    return sum(filter(lambda x: x % 2 == 0, my_list[29:80]))
```
This function takes a list of integers as an argument and returns the sum of all even integers from index 29 to index 79, both inclusive. If no even integers exist in the specified range, the function will return 0.

For example, if `my_list = [1, 2, 3, 4, 5, 6, 7, 8, 9]`, then `sum_even_ints_inclusive(my_list)` will return `2 + 4 + 6 + 8 = 20`.

Note that this function uses the built-in `filter()` function to remove non-even integers from the list, and the `sum()` function to calculate the sum of the remaining even integers.
