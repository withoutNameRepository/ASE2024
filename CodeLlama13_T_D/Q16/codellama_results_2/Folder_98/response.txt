```
def sum_even_ints_inclusive(my_list):
    result = 0
    for i in range(len(my_list)):
        if my_list[i] % 2 == 0 and i <= 6:
            result += my_list[i]
    return result
```
This function takes a list of integers as an argument, iterates through the list using a for loop, and checks each element to see if it is even. If an even integer is found in the range from index 0 to index 6, both inclusive, its value is added to the variable 'result'. Finally, the function returns the value of result.
