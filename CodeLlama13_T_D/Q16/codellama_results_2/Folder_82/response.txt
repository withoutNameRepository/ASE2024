```
def sum_even_ints_inclusive(my_list):
    return sum(x for i, x in enumerate(my_list) if i >= 29 and i <= 29 and x % 2 == 0)
```
This function uses a list comprehension to iterate over the elements of `my_list`, starting from index 29 (both inclusive), and sum up all even integers in the range. If no even integers exist in the specified range, the function returns 0.
