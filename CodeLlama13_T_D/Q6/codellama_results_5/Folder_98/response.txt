```python
def all_neg_ints_exclusive(my_list):
    return [x for x in my_list if x < 0 and 0 <= x <= 6]
```
This function uses a list comprehension to iterate over the elements of `my_list`, and returns a new list that contains all the negative integers from index 0 to index 6, both exclusive. If no negative integers exist in the specified range, an empty list is returned.
