```python
def all_neg_ints_exclusive(my_list):
    return [x for x in my_list[36:54] if x < 0]
```
Note that this function uses a list comprehension to create the new list of negative integers. It starts by iterating over the slice of `my_list` from index 36 to index 54 (exclusive). For each element `x`, it checks whether `x` is less than 0, and if so, it includes it in the resulting list. If no such elements exist, the list comprehension will return an empty list.
