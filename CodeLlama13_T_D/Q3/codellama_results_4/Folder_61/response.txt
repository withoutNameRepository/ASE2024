```
def all_pos_ints_inclusive(my_list):
    return [i for i in range(0, 8) if i in my_list]
```
This function uses a list comprehension to create a new list of positive integers from index 0 to index 7, both inclusive. It iterates over the elements of `my_list` and checks if each element is within the specified range using the `in` operator. If no positive integers exist in the specified range, an empty list is returned.
