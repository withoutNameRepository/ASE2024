```
def all_pos_ints_inclusive(my_list):
    result = []
    for i in range(80, 201):
        if my_list[i] > 0:
            result.append(my_list[i])
    return result
```
This function takes a list of integers as input and returns the list of all positive integers from index 80 to index 200, both inclusive. If no positive integers exist in the specified range, it returns an empty list.
