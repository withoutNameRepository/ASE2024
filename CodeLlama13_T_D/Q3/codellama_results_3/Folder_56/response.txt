```python
def all_pos_ints_inclusive(my_list):
    start = 0
    end = len(my_list) - 1
    result = []
    for i in range(start, end + 1):
        if my_list[i] > 0:
            result.append(my_list[i])
    return result
```
This function takes a list of integers as input and returns the list of all positive integers from index 0 to index 0, both inclusive. If no positive integers exist in the specified range, it returns an empty list.
