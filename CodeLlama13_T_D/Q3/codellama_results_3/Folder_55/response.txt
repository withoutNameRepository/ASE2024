```
def all_pos_ints_inclusive(my_list):
    return [i for i in range(0, 11) if i in my_list]
```
This function uses a list comprehension to generate the list of positive integers from index 0 to index 10 (both inclusive), using the `range()` function and checking if each integer is in the input list. If no positive integers exist in the specified range, an empty list is returned.
