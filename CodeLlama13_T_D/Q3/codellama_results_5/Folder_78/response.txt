```python
def all_pos_ints_inclusive(my_list):
    return [x for x in my_list if 43 <= x <= 51 and x > 0]
```
This function uses a list comprehension to iterate over the input `my_list`, and returns a new list containing only positive integers from index 43 to index 51, both inclusive. If no positive integers exist in the specified range, the function returns an empty list.
