def find_n_th_smallest_num(distinct_numbers):

Note that the `sorted()` function returns a new list with all the elements of the input list, sorted in ascending order. The `[12:90]` slice selects the elements from index 12 to index 90 (both inclusive), and the `[16]` index selects the 17th element of that slice.
