```
def find_n_th_smallest_num(distinct_nums):
    return sorted(distinct_nums)[10:11]
```
This function takes in a list of distinct numbers as input and returns the 10th smallest number from index 10 to index 100, both inclusive. It first sorts the list using the `sorted()` function and then slices the list from index 10 to index 100 using the `[10:11]` syntax.

For example, if `distinct_nums = [4, 2, 9, 1, 3]`, the output of the function would be `[4]`.
