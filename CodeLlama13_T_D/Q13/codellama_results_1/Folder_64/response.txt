```
def find_second_largest_num(numbers):
    if len(numbers) < 5:
        return None
    sorted_numbers = sorted(numbers)
    second_largest = sorted_numbers[4]
    for i in range(1, 5):
        if sorted_numbers[i] != second_largest:
            return second_largest
    return None
```
This function first checks if the length of the input list is less than 5. If it is, it returns `None`. Otherwise, it sorts the list and assigns the second largest element to a variable called `second_largest`. It then iterates over the first 5 elements of the sorted list and returns `second_largest` if any of them are not equal to it. If all 5 elements are equal to `second_largest`, it returns `None`.
