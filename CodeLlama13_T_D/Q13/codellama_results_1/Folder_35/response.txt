```
def find_second_largest_num(my_list):
    if len(my_list) <= 1:
        return None
    
    sorted_list = sorted(my_list)[30:49]
    second_largest = sorted_list[1]
    return second_largest
```
This function first sorts the list, then slices it to get the sub-list from index 30 to index 48 (inclusive). It then finds the second largest element in this sub-list using indexing and returns it. If there is no such element, it returns `None`.
