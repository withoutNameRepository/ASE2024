```python
def find_second_largest_num(my_list):
    if len(my_list) < 2:
        return None
    
    sorted_list = sorted(my_list)[1:-1]
    if len(sorted_list) == 0:
        return None
    
    second_largest_num = sorted_list[len(sorted_list) // 2]
    return second_largest_num
```
This function first checks the length of the input list. If it is less than 2, there is no second largest element, so the function returns `None`. Otherwise, it sorts the list and then takes the middle element (using integer division) as the second largest number. This ensures that if there are an odd number of elements in the list, the middle element is the second largest one. Finally, the function returns this value or `None` if the length of the sorted list is 0.
