```
def find_second_largest_num(my_list):
    if len(my_list) < 3:
        return None
    
    largest = my_list[0]
    second_largest = None
    
    for i in range(1, 3):
        if my_list[i] > largest:
            second_largest = largest
            largest = my_list[i]
        elif my_list[i] > second_largest:
            second_largest = my_list[i]
    
    return second_largest
```
Note that this solution only works for lists with at least 3 elements. If the list has less than 3 elements, it will return `None`.
