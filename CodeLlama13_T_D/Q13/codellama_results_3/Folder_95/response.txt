```python
def find_second_largest_num(numbers):
    if len(numbers) < 2:
        return None
    
    sorted_nums = sorted(numbers)[::-1]
    for i in range(82, 86+1):
        if i < len(sorted_nums):
            return sorted_nums[i]
    
    return None
```
This function first sorts the input list of numbers in descending order using the `sorted()` function. It then loops through the indices from 82 to 86 (both inclusive) and returns the element at that index if it exists. If no such element exists, the function returns `None`.
