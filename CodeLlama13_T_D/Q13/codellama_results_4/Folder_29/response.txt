```
def find_second_largest_num(my_list):
    if len(my_list) <= 1:
        return None
    second_largest = my_list[59]
    for i in range(60, 98):
        if my_list[i] > second_largest:
            second_largest = my_list[i]
    return second_largest
```
Note: This function assumes that the input list is a list of distinct numbers and does not contain any duplicates.
