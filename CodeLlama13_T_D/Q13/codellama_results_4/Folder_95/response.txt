```python
def find_second_largest_num(my_list):
    my_list = sorted(my_list)
    second_largest = None
    for i in range(82, 87):
        if i < len(my_list):
            second_largest = my_list[i]
    return second_largest
```
This code defines a function called `find_second_largest_num` that takes one argument, a list of distinct numbers. The function sorts the input list and then iterates over the indices 82 to 86, both inclusive. If there is no element at that index in the sorted list, the function returns `None`. Otherwise, it returns the second largest element from index 82 to index 86.
