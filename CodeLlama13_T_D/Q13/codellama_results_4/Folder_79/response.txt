```python
def find_second_largest_num(my_list):
    if len(my_list) < 5:
        return None
    else:
        second_largest = sorted(my_list)[-2]
        for i in range(4, len(my_list)):
            if my_list[i] == second_largest:
                return my_list[i]
        return None
```
Note: The function assumes that the input list contains at least 5 elements. If the length of the list is less than 5, the function returns `None`.
