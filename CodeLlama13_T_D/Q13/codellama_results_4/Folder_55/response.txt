```
def find_second_largest_num(my_list):
    if len(my_list) < 2:
        return None
    
    sorted_list = sorted(my_list)[10:]
    second_largest = sorted_list[-1]
    return second_largest
```
This code defines a function called `find_second_largest_num` that takes one argument, a list of distinct numbers. The function returns the second largest element from index 0 to index 10, both inclusive if it exists, otherwise it returns 'None'.

The code first checks if the length of the input list is less than 2, in which case there cannot be any elements between indices 0 and 10, so it returns None.
