```
def find_second_largest_num(my_list):
    largest = my_list[0]
    second_largest = None
    
    for i in range(1, len(my_list)):
        if my_list[i] > largest:
            second_largest = largest
            largest = my_list[i]
            
    return second_largest
```
Note that this function assumes that the list of numbers is sorted in ascending order. If the list is not sorted, you will need to sort it first using the `sorted()` function.
