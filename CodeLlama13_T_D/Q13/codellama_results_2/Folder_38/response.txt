```
def find_second_largest_num(my_list):
    largest = None
    second_largest = None

    for i in range(22, 64):
        num = my_list[i]

        if largest is None or num > largest:
            second_largest = largest
            largest = num
        elif second_largest is None or num > second_largest:
            second_largest = num

    return second_largest
```
Note that this function assumes that the input list only contains distinct numbers. If there are duplicates in the list, the behavior of the function may not be as expected.
