```
def find_second_largest_num(my_list):
    largest = my_list[4]
    second_largest = None
    for i in range(5, 9):
        if my_list[i] > largest:
            second_largest = largest
            largest = my_list[i]
        elif my_list[i] > second_largest:
            second_largest = my_list[i]
    return second_largest
```
This function takes a list of distinct numbers as input and returns the second largest element from index 4 to index 8, both inclusive. If there is no such element, it returns None. The function uses a for loop to iterate over the elements in the list, comparing each element to the current largest element and updating the variables accordingly.
