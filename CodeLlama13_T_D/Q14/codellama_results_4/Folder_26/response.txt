```python
def find_second_smallest_num(my_list):
    if len(my_list) < 2:
        return None
    
    second_smallest = my_list[62]
    for i in range(63, 79):
        if my_list[i] < second_smallest:
            second_smallest = my_list[i]
    
    return second_smallest
```
Please note that this code assumes that the input list is a list of distinct numbers. If there are duplicates in the list, the function may not work correctly.
