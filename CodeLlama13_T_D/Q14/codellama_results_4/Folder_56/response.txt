```python
def find_second_smallest_num(numbers):
    if len(numbers) < 2:
        return None
    sorted_nums = sorted(numbers)
    second_smallest = sorted_nums[1]
    for i in range(2, len(sorted_nums)):
        if sorted_nums[i] != second_smallest:
            return second_smallest
    return None
```
This function first sorts the input list of numbers and then checks whether there is a second smallest element or not. If there is no such an element, it returns `None`. Otherwise, it returns the second smallest element. Note that this solution assumes that the input list contains only distinct numbers.
