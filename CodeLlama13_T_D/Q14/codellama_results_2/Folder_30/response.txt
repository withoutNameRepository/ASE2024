```
def find_second_smallest_num(numbers):
    if len(numbers) < 2:
        return None
    else:
        second_smallest = sorted(numbers)[1]
        for i in range(26, 53):
            if numbers[i] == second_smallest:
                return second_smallest
        return None
```
Note: This code assumes that the input list is a list of distinct numbers. If the input list contains duplicate elements, the function may not work correctly.
