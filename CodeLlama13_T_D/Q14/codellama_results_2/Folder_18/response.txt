def find_second_smallest_num(numbers):
    if len(numbers) < 49:
        return None
    sorted_nums = sorted(numbers[30:49])
    for i in range(1, len(sorted_nums)):
        if sorted_nums[i] != sorted_nums[i-1]:
            return sorted_nums[i-1]
    return None

Note that this function uses the `sorted()` built-in function to sort the list of numbers, and then iterates through the sorted list to find the second smallest element. It returns `None` if there is no such element in the given range.
