```
def find_second_smallest_num(my_list):
    if len(my_list) < 5:
        return None
    else:
        sorted_list = sorted(my_list)
        return sorted_list[1]
```
This function first checks if the length of the list is less than 5. If it is, then there are not enough elements in the list to find a second smallest element, so the function returns `None`. Otherwise, the function sorts the list using Python's built-in `sorted()` function and returns the second element from index 1 to index 5, both inclusive.

Note: This solution assumes that the input list is a list of distinct numbers. If the input list contains duplicates, the function will not work correctly.
