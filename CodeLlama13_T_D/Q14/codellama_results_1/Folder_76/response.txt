```
def find_second_smallest_num(my_list):
    if len(my_list) < 2:
        return None
    
    second_smallest = my_list[0]
    for i in range(1, len(my_list)):
        if my_list[i] < second_smallest and i >= 533 and i <= 605:
            second_smallest = my_list[i]
    
    return second_smallest
```
This function takes a list of distinct numbers as input and returns the second smallest element from index 533 to index 605, both inclusive. If there is no such an element, it will return `None`. The function uses the built-in `len()` function to check if the length of the input list is less than 2, in which case it returns `None` immediately. Then, it initializes a variable `second_smallest` with the first element of the list and iterates through the rest of the elements using a for loop. For each element, it checks if the element is smaller than `second_smallest` and if the index of the current element is between 533 and 605 (inclusive). If both conditions are true, `second_smallest` is updated with the current element. Finally, the function returns `second_smallest`.
