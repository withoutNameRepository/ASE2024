```python
def find_second_smallest_num(nums):
    if len(nums) < 2:
        return None
    sorted_nums = sorted(nums)
    for i in range(1, len(sorted_nums)):
        if sorted_nums[i] != sorted_nums[i-1]:
            return sorted_nums[i]
    return None
```
This solution uses the `sorted()` function to sort the input list of numbers, and then loops through the list starting from the second element (index 1) to find the first element that is not equal to the previous element. If there are no such elements, it returns `None`.
