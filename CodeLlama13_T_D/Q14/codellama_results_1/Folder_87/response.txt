```python
def find_second_smallest_num(my_list):
    if len(my_list) <= 2:
        return None
    
    sorted_list = sorted(my_list[22:89])
    second_smallest = sorted_list[1]
    return second_smallest
```
This function sorts the list and then returns the second smallest element from index 22 to index 88. If there is no such an element, it returns None.
