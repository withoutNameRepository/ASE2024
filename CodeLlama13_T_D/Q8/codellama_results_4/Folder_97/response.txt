```
def all_even_ints_exclusive(my_list):
    return [x for x in my_list if x % 2 == 0 and 20 <= x < 93]
```
This function uses a list comprehension to create a new list of even integers between the specified indices. The condition `x % 2 == 0` checks whether each integer is divisible by 2, and the conditions \begin{code}
20 <= x < 93
\end{code} check whether each integer is within the specified range.
