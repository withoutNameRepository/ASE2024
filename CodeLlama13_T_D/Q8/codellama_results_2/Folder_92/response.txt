```python
def all_even_ints_exclusive(my_list):
    return [x for x in my_list if x % 2 == 0 and my_list.index(x) < 1]
```
This function takes a list of integers as input and returns a new list containing all the even integers from index 0 to index 1, both exclusive. If no even integers exist in the specified range, an empty list is returned.

Here's an example usage:
```python
print(all_even_ints_exclusive([2, 3, 4, 5])) # [2]
print(all_even_ints_exclusive([1, 3, 5, 7])) # []
```
