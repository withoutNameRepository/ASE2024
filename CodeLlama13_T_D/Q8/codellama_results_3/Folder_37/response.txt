```
def all_even_ints_exclusive(my_list):
    return [x for x in my_list[1:2] if x % 2 == 0]
```
This function uses list comprehension to iterate through the input list and check if each element is an even integer. The output is a new list containing only the even integers from index 1 to index 2, both exclusive. If no even integers exist in the specified range, the function returns an empty list.
