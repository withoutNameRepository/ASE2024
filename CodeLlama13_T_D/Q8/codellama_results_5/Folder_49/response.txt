```python
def all_even_ints_exclusive(nums):
    return [x for x in nums[80:200] if x % 2 == 0]
```
Note: This function uses a list comprehension to create a new list of even integers from the input list, using the modulo operator (%) to check if each integer is even. The `x for x in nums[80:200]` syntax specifies that we want to iterate over the elements of `nums` from index 80 to index 200, both exclusive.
