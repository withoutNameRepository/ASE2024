```
def all_even_ints_exclusive(my_list):
    return [x for x in my_list[10:75] if x % 2 == 0]
```
This function takes a list of integers as input and returns a new list containing only the even integers from index 11 to index 76, both exclusive. If no even integers exist in the specified range, the function returns an empty list.

Here is an example usage of the function:
```
my_list = [3, 5, 2, 8, 10, 9, 15]
print(all_even_ints_exclusive(my_list)) # Output: [8, 10]
```
