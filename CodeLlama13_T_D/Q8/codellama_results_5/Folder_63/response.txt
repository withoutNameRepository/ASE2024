```python
def all_even_ints_exclusive(numbers):
    return [num for i, num in enumerate(numbers) if i > 37 and i < 53 and num % 2 == 0]
```
This function uses a list comprehension to iterate over the input list `numbers`, and returns a new list containing all even integers from index 38 to index 52, both exclusive. If no even integers exist in the specified range, an empty list is returned.
