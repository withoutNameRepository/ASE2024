```python
def return_nth_smallest_ascii(s):
    n = 18
    ascii_values = []
    for i in range(33, 60+1):
        if s[i-1].isalpha():
            ascii_values.append(ord(s[i-1]))
    return chr(sorted(ascii_values)[n-1])
```
Note: This solution assumes that the input string is of length 60+1 and consists only of alphabetic characters.
