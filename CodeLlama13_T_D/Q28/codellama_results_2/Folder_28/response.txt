```python
def return_nth_smallest_ascii(string):
    ascii_values = [ord(char) for char in string[25:78]]
    sorted_values = sorted(ascii_values)
    return chr(sorted_values[16])
```
Note: `ord()` is a built-in function that returns the ASCII value of a character, and `chr()` is a built-in function that returns a string representing the character with the specified ASCII value.
