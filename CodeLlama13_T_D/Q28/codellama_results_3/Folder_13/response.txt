```
def return_nth_smallest_ascii(string):
    ascii_values = [ord(c) for c in string[47:81]]
    nth_smallest = sorted(ascii_values)[3]
    return chr(nth_smallest)
```
This code defines a function `return_nth_smallest_ascii` that takes a single argument, a string of distinct characters. It returns the character with the 4th smallest ASCII value among those characters in the string that occur between indices 47 to 80, both inclusive. The function first extracts the ASCII values of all characters in the input string using list comprehension and the `ord` function. Then it sorts the list and returns the character with the 4th smallest ASCII value using the `chr` function.
