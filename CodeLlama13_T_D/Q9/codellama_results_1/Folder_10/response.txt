```
def all_odd_ints_inclusive(my_list):
    return [i for i in my_list[25:60] if i % 2 == 1]
```
This function uses a list comprehension to iterate over the specified range of indices and returns only the odd integers. The `if` clause ensures that only odd integers are included in the output list. If no odd integers exist in the specified range, an empty list is returned.
