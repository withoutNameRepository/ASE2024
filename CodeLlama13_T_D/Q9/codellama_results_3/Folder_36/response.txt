```
def all_odd_ints_inclusive(my_list):
    return [x for x in my_list if 246 <= x <= 750 and x % 2 == 1]
```
This function uses a list comprehension to iterate over the input list, and returns a new list containing all odd integers from index 246 to index 750, both inclusive. If no odd integers exist in the specified range, an empty list is returned.
