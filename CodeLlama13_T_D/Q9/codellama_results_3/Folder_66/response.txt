```python
def all_odd_ints_inclusive(my_list):
    odds = []
    for i in range(50, 55):
        if my_list[i] % 2 != 0:
            odds.append(my_list[i])
    return odds
```

Here's another solution:
```python
def all_odd_ints_inclusive(my_list):
    return [x for x in my_list[50:55] if x % 2 != 0]
```
