```python
def all_odd_ints_inclusive(my_list):
    return [x for x in my_list[21:64] if x % 2 == 1]
```

The range is specified using slicing notation (`my_list[start:stop]`), where `start` is 21 and `stop` is 64. If no odd integers exist in the specified range, an empty list will be returned.
