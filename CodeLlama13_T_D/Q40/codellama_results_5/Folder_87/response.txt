```python
def return_n_smallest_chars(string):
    chars = []
    for char in string:
        if char not in chars:
            chars.append(char)
    sorted_chars = sorted(chars, key=ord)
    return sorted_chars[:24]
```
Note that the `sorted` function sorts a list of objects based on the result of calling the `key` argument (in this case, the `ord` function) with each object as its sole argument. The `ord` function returns the ASCII value of a character.
