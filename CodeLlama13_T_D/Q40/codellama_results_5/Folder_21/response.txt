```
def return_n_smallest_chars(string):
    ascii_list = sorted([ord(char) for char in string])
    return [''.join(chr(ascii_value) for ascii_value in ascii_list[:44]])
```
This function uses a list comprehension to create a list of the ASCII values of the characters in the input string. It then sorts this list and joins it back into a string using `chr` and `join`. Finally, it returns the resulting 44-character string.
