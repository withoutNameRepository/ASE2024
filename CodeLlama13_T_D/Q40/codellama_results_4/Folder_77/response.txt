```
def return_n_smallest_chars(s):
    chars = sorted([c for c in s if c.isalpha()], key=lambda x: ord(x))[:47]
    return chars
```
This code defines a function called `return_n_smallest_chars` that takes one argument, `s`, which is a string of distinct characters. The function first creates a list of all the alpha characters in the input string using list comprehension and sorted in ascending order based on their ASCII values using the `key` parameter of the sorted function.

Finally, the function returns the first 47 elements from this sorted list.
