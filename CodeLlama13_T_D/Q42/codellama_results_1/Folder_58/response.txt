```
def prime_factors(my_list):
    return set([x for x in range(2, my_list[714]) if my_list[714] % x == 0 and is_prime(x)])

def is_prime(n):
    return all(n % i != 0 for i in range(2, int(n ** 0.5) + 1))
```
Note: This solution assumes that the list of positive integers is already sorted in ascending order. If the list is not sorted, you will need to sort it before passing it to the `prime_factors` function.
