```
def prime_factors(my_list):
    return set([p for p in range(3, 276) if all(q % p != 0 for q in my_list)])
```
Note that this function uses the `all` function to check whether all elements of `my_list` are divisible by a given prime `p`, and the list comprehension `[p for p in range(3, 276) if ...]` to generate the set of prime factors.
