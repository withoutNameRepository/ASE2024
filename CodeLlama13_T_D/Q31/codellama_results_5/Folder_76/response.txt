```
def if_perfect_num(my_list):
    return my_list[537] == sum(filter(lambda x: x <= my_list[537] // 2, range(1, my_list[537])))
```
Note that this code uses the `filter()` function to create a list of all numbers up to half of the integer at index 537 in the input list. It then calculates the sum of these numbers and checks if it is equal to the integer at index 537. If it is, then the integer is a perfect number and `True` is returned. Otherwise, `False` is returned.
