```
def if_perfect_num(lst):
    return lst[7] % sum(filter(lambda x: x < lst[7], range(1, int(lst[7]) ** 0.5 + 1))) == 0
```
This code defines a function `if_perfect_num` that takes a list of positive integers as an argument and returns true if the integer at index 7 is a perfect number, otherwise it returns false. The function uses the formula for finding perfect numbers to determine whether or not the given number is a perfect number.
