```python
def all_left_right_truncatable_prime(x):
    primes = []
    for i in range(10, x+1):
        if is_prime(i) and not contains_zero(str(i)):
            primes.append(i)
    return primes
```
This function uses the `is_prime` function to check whether a number is prime or not, and the `contains_zero` function to check whether a string representation of the number contains any zeros. It then returns the list of all left-and-right-truncatable prime numbers less than or equal to `x`.
