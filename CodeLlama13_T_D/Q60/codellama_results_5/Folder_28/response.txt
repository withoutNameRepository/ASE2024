```python
def all_left_right_truncatable_prime(x):
    primes = []
    for i in range(10, x+1):
        if is_prime(i) and not contains_zero(str(i)):
            primes.append(i)
    return sorted(primes)
```
Note: The `is_prime` function checks whether the given integer is a prime number or not, and the `contains_zero` function checks whether the given string (representation of an integer) contains any zeroes.
