```python
def all_left_right_truncatable_prime(x):
    primes = []
    for i in range(2, x+1):
        if is_prime(i) and not contains_zero(str(i)):
            primes.append(i)
    return sorted(primes, reverse=True)
```
Note: The `is_prime` function can be implemented using a simple trial division algorithm, or by checking the primality of the number modulo various small prime numbers (e.g. 2, 3, 5, 7, etc.). The `contains_zero` function can be implemented by checking if the string representation of the number contains any '0' characters.
