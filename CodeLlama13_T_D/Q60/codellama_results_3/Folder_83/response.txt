```python
def all_left_right_truncatable_prime(x):
    left_and_right_truncatable_primes = []
    for i in range(10, x+1):
        if is_prime(i) and no_zeroes(str(i)):
            left_and_right_truncatable_primes.append(i)
    return sorted(left_and_right_truncatable_primes)
```
The `is_prime` function checks whether a number is prime or not using the sieve of Eratosthenes algorithm, and the `no_zeroes` function checks whether a string contains any zeroes. The `sorted` function sorts the list in ascending order.
