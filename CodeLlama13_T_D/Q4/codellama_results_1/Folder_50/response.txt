```python
def all_pos_ints_exclusive(my_list):
    return [x for x in my_list if 70 < x < 98 and x > 0]
```
This function takes a list of integers as input, `my_list`, and returns a new list containing all positive integers from index 70 to index 98, both exclusive. If no positive integers exist in the specified range, an empty list is returned.

Here's an example usage:
```python
my_list = [1, 2, 3, -4, 5, 6, 7, 8, 9]
print(all_pos_ints_exclusive(my_list)) # Output: [7, 8, 9]
```
