```
def all_pos_ints_exclusive(my_list):
    return [x for x in my_list if x > 50 and x < 92 and x % 1 == 0]
```
This code defines a function called `all_pos_ints_exclusive` that takes one argument, a list of integers, and returns the list of all positive integers from index 50 to index 92, both exclusive. The code uses a list comprehension with an if statement to filter out any non-integer or negative values in the input list, and then returns the resulting list. If no positive integers exist in the specified range, the function returns an empty list.
