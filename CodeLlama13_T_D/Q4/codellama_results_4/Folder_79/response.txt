```
def all_pos_ints_exclusive(my_list):
    return [x for x in my_list if 3 < x < 8 and x > 0]
```
This function uses a list comprehension to iterate over the elements of `my_list` and selects those that are within the specified range of 3 and 8, both exclusive. The function returns an empty list if no positive integers exist in this range.
