```python
def all_pos_ints_exclusive(my_list):
    return [x for x in my_list if x > 37 and x < 53 and x % 1 == 0]
```
This function takes a list of integers as an argument, checks whether each element is a positive integer within the specified range (38 to 52, both exclusive), and returns a new list containing all the positive integers from that range. If no positive integers exist in the specified range, the function returns an empty list.

Here's an example usage of this function:
```python
>>> my_list = [1, 2, 3, 4, 5]
>>> all_pos_ints_exclusive(my_list)
[3, 4]
```
