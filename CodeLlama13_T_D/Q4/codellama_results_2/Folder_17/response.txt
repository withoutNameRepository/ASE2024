```python
def all_pos_ints_exclusive(arr):
    return [x for x in arr if 62 < x < 96 and x > 0]
```
This function takes a list of integers as input, `arr`, and returns a new list containing only the positive integers from index 62 to index 96, both exclusive. The function uses a list comprehension to create a new list containing only the elements that meet the specified criteria.
