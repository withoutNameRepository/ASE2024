```python
def all_pos_ints_exclusive(my_list):
    return [i for i in range(1, 6) if i in my_list]
```
This function uses a list comprehension to create a new list of positive integers from index 0 to index 5 (both exclusive). The `range` function is used to generate the numbers, and the `if i in my_list` clause filters out any negative or zero values. If no positive integers exist in the specified range, an empty list is returned.

Note that this function will also work for other lists of integers, not just the one specified in the problem description.
