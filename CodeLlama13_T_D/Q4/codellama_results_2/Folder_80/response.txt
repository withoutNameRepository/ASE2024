```python
def all_pos_ints_exclusive(my_list):
    return [i for i in my_list if 69 < i < 97 and i > 0]
```
This function uses a list comprehension to iterate over the input list `my_list` and returns only those elements that are positive integers between index 69 and index 97, both exclusive. If no such integer exists in the specified range, an empty list is returned.
