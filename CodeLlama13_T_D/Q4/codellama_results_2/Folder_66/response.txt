```
def all_pos_ints_exclusive(my_list):
    return list(filter(lambda x: x > 12 and x < 76, my_list))
```
This function uses the `filter()` function to iterate over the elements of the input list and keep only those that are greater than 12 and less than 76. The resulting filtered list is then converted back to a list using the `list()` function. If no positive integers exist in the specified range, the function returns an empty list.
