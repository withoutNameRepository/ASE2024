```python
def all_pos_ints_exclusive(my_list):
    return [i for i in range(1, len(my_list)) if my_list[i-1] < my_list[i]]
```
This code uses list comprehension to iterate over the indexes of `my_list` and check whether the corresponding value is a positive integer. The resulting list only contains the indices of the positive integers, which are then used to create a new list using `range()`. The function returns an empty list if no positive integers exist in the specified range.
