```
def all_pos_ints_exclusive(my_list):
    return [i for i in range(5) if i in my_list and i > 0]
```
This function uses a list comprehension to create a new list of positive integers from the specified range. The `range` function is used to generate the indices, and the `if` clause is used to filter out any negative or zero values. Finally, the resulting list is returned.
