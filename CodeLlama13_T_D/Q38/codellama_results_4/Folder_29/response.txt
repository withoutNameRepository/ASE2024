```python
def find_subset_of_length_n(elements, n):
    return len([subset for subset in powerset(elements) if len(subset) == n])
```
This is a simple function that uses a list comprehension to iterate over the power set of `elements` and returns the number of subsets with size equal to `n`. The function uses the built-in `powerset()` function to generate all possible subsets of `elements`.
