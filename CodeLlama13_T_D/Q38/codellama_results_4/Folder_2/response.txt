```
def find_subset_of_length_n(elements, n):
    subsets = []
    for i in range(len(elements)):
        if len(subsets) == n:
            break
        subset = set()
        for j in range(i+1, len(elements)):
            subset.add(elements[j])
            subsets.append(subset)
    return subsets
```
Note that this code assumes that the input `elements` is a set of unique elements. If the input contains duplicate elements, the code will produce duplicates in the output.
