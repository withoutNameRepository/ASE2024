```
def find_subset_of_length_n(my_set, n):
    return len([subset for subset in powerset(my_set) if len(subset) == n])
```
This function uses the `powerset` function from the `itertools` module to generate all subsets of a given set. Then it filters this list by checking if the length of each subset is equal to the given size `n`. Finally, it returns the number of these filtered subsets.
