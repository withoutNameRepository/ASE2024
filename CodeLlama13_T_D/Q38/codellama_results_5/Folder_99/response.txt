```
def find_subset_of_length_n(elements, n):
    return len([subset for subset in powerset(elements) if len(subset) == n])
```
[SYS]> The code uses the `powerset` function from the `itertools` module to generate all subsets of the input set and then filters them by size. The length of the filtered list is returned as the result. [/SYS]
