```
def all_neg_ints_inclusive(my_list):
    return [x for x in my_list[17:79] if x < 0]
```
Explanation:

* `my_list` is the input list of integers.
* `my_list[17:79]` is a slice of the list from index 17 to index 78, both inclusive.
* `[x for x in my_list[17:79] if x < 0]` is a list comprehension that creates a new list containing all negative integers in the specified range. The `if` clause filters out any non-negative integers.

This code returns an empty list if no negative integers exist in the specified range, otherwise it returns a list of all negative integers from index 17 to index 78, both inclusive.
