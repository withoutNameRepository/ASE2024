```
def all_neg_ints_inclusive(my_list):
    return [x for x in my_list if 0 <= x < 5 and x < 0]
```
This function takes a list of integers as an argument, and returns a new list containing all negative integers from index 0 to index 4 (both inclusive) that are present in the original list. If no negative integers exist in the specified range, the function returns an empty list.

For example, if we call the function with `my_list = [-2, -1, 0, 1, 2]`, it will return `[-2, -1]`. If we call the function with `my_list = [3, 4, 5, 6, 7]`, it will return an empty list.
