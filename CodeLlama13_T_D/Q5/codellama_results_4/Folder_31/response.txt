```python
def all_neg_ints_inclusive(my_list):
    return [x for x in my_list[75:89] if x < 0]
```
This function uses a list comprehension to iterate over the specified range of the input list, and returns only the elements that are negative. The `if x < 0` condition is used to filter out any non-negative integers in the range. If no negative integers exist in the specified range, an empty list will be returned.
