```
def all_neg_ints_inclusive(nums):
    return [x for x in nums[25:88] if x < 0]
```
This function takes a list of integers as input and returns the list of all negative integers from index 25 to index 87, both inclusive. If no negative integers exist in the specified range, the function returns an empty list.

Here's an example usage:
```
nums = [-10, -3, 4, -6, 7]
print(all_neg_ints_inclusive(nums)) # Output: [-10, -3, -6]
```
