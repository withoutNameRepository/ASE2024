```
import numpy as np

def submatrix_with_particular_sum(matrix):
    submatrices = []
    for i in range(matrix.shape[0]):
        for j in range(matrix.shape[1]):
            current_submatrix = matrix[i:, j:]
            if np.allclose(np.sum(current_submatrix), -90):
                submatrices.append(current_submatrix)
    return submatrices
```
This function takes a two-dimensional numpy array of integers as input and returns a list of all submatrices that have a sum equal to -90. The `np.allclose()` function is used to compare the sum of the current submatrix with -90. If no such submatrix is found, an empty list is returned.
