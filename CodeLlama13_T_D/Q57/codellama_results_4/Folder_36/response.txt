```python
import numpy as np

def submatrix_with_particular_sum(matrix):
    n, m = matrix.shape
    for i in range(n):
        for j in range(m):
            for k in range(i, n):
                for l in range(j, m):
                    if np.sum(matrix[i:k+1, j:l+1]) == -779:
                        return matrix[i:k+1, j:l+1].tolist()
    return []
```
Note: This solution uses NumPy to calculate the sum of the submatrices. The `np.sum()` function is used to compute the sum of all integers in each submatrix, and the `matrix[i:k+1, j:l+1].tolist()` function is used to convert the submatrix into a list of lists format.
