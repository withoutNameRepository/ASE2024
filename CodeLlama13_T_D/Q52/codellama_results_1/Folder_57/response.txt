```python
def palindrome_of_length_n(s):
    return set([s[i:i+92] for i in range(len(s)-91) if s[i:i+92].lower() == s[i:i+92][::-1].lower()])
```
This function takes a string `s` as input and returns a set of all palindromes of length 92 that are substrings of `s`. The palindromes are found case-insensitively, meaning that the string is converted to lowercase before checking for palindromes. If there are no palindromes of the specified length as substrings of the given string, the function returns an empty set.
