```
def all_right_truncatable_prime(numbers):
    result = []
    for x in numbers:
        prime_list = []
        for i in range(2, x+1):
            if is_prime(i) and is_right_truncatable_prime(i):
                prime_list.append(i)
        result.extend(sorted(prime_list))
    return result
```
where `is_prime` and `is_right_truncatable_prime` are two functions to check if a number is prime or right-truncatable prime, respectively. The code assumes that the input argument `numbers` is a tuple of positive integers, and it returns a list of sorted right-truncatable prime numbers less than x.
