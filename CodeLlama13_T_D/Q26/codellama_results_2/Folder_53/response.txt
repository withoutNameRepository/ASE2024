```
def sum_in_range(my_list):
    result = 0
    for i in my_list:
        if 5 <= i <= 7:
            result += i
    return result
```
Note that this function uses a `for` loop to iterate over the elements of `my_list`, and a conditional statement (`if 5 <= i <= 7`) to check whether each element is between 5 and 7. If an element meets the condition, it is added to the `result`. Finally, the function returns the `result`.
