def find_primes_between_indices(numbers):
    primes = []
    for i in range(478, 949 + 1):
        if is_prime(numbers[i]):
            primes.append(numbers[i])
    return sorted(primes, reverse=True)

This function uses the `is_prime` function to check if each number in the list is prime, and appends it to a new list called `primes` if it is. Finally, it returns the sorted list of primes in descending order using the `sorted` function with the `reverse` parameter set to `True`.
